Lezione 02: Inline Style

CSS

  • CSS significa Cascading Style Sheets, cioè fogli di stile a cascata (che indica che lo stile viene applicato dall'alto al basso)
  • CSS descrive come gli elementi di un documento HTML vengono visualizzati (colore, dimensione, font, bordo, ...)
