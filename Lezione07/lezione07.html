A

Una delle funzionalità principali e storiche dell'HTML e del WWW sono i link (ipertesto). Per inserire un link usa il tag a

Il tag a significa anchor è crea un link ad una pagina web

Oppure un link ad una sezione della pagina

E' anche possibile specificare dove aprire il link "cliccato" tramite l'attributo target.

Apri il link in una nuova tab del browser.

Ma ovviamente un'immagine vale 1000 parole.

Per inserire un immagine usa il tag img. L'attributo src serve per specificare l'indirizzo o URL dell'immagine.

L'immagine può essere sul Web o sul Disco locale

E' sempre corretto aggiungere l'attributo alt per inserire un testo alternativo in caso in cui l'immagine non sia disponibile.
