DEFINITION LIST

In un testo o un articolo, risulta particolarmente utile organizzare il contenuto in modo schematico e rappresentare le informazioni tramite una lista.

HTML ci viene in aiuto con un definition list tramite tre tag dl, dt e dd

dl
Rappresenta un definition list
dt
Rappresenta un definition term
dd
Rappresenta un item o la definizione di un termine

UNORDERED LIST

Ci sono poi le liste di tipo non ordinato che si creano con 2 tag:

  • ul: tag che definisce la lista non ordinata Unordered List
  • li: tag che definisce l'elemento/item della lista List Item

ORDERED LIST

Ci sono poi le liste di tipo ordinato che si creano con 2 tag:

  1. ol: tag che definisce la lista ordinata Ordered List
  2. li: tag che definisce l'elemento/item della lista List Item
